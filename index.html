Résultat - Question 1

Les classes grammaticales

Correct !

Le mot "attentivement" est bien un adverbe.

Explication :

Un adverbe est un mot invariable qui modifie ou précise le sens :

  • d'un verbe : "Elle lit attentivement."
  • d'un adjectif : "Un livre très intéressant."
  • d'un autre adverbe : "Elle lit très attentivement."

Les adverbes se terminent souvent par le suffixe -ment, formé à partir d'un adjectif (comme "attentif" → "attentivement").

Dans notre phrase, "attentivement" précise la manière dont l'action de lire est effectuée.

Résultat - Question 1

Les classes grammaticales

Incorrect

Le mot "attentivement" n'est pas un adjectif qualificatif mais un adverbe.

Analyse de l'erreur :

Vous avez confondu deux classes grammaticales importantes :

  • Adjectif qualificatif : Qualifie un nom, s'accorde en genre et en nombre avec ce nom (ex: "la petite fille")
  • Adverbe : Mot invariable qui modifie un verbe, un adjectif ou un autre adverbe (ex: "Elle lit attentivement")

Dans notre phrase, "attentivement" précise la manière dont l'action de lire est effectuée. Il est invariable et se rapporte au verbe, c'est donc un adverbe.
